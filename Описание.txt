В лабораторной работе 3 были созданы четыре класса для выполнения различных логических бинарных операций: XOR, OR, AND, штрих Шеффера.
Данные класссы являются наследниками класса Logic Operations и в каждом из них переопределяются методы getSign, getName и estimate, позволяющие 
вывести знак, буквенное обозначение и результат операции соответственно. Использование одних и тех же методов различных подклассов позволяет 
обеспечить полиморфизм. Входные данные для операций - логический ноль или логическая единица(false или true) задаются в методе main класса UI.
Пример:
И
false * true = false
ИЛИ
false + true = true
Штрих Шеффера
false | true = true
Исключающее ИЛИ
false xor true = true
