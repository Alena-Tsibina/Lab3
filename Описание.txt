В лабораторной работе 3 были созданы четыре класса для выполнения различных логических бинарных операций: XOR, OR, AND, штрих Шеффера, которые 
относятся к интерфейсу LogicOperations. В каждом из них по разному определяются методы, объявленные в классе LogicOperations: getSign, getName 
и estimate позволяющие вывести знак, буквенное обозначение и результат операции соответственно. Разное определение одних и тех же методов в 
различных классах интерфейса LogicOperations позволяет обеспечить полиморфизм. Входные данные для операций - логический ноль или логическая 
единица(false или true) задаются в методе main класса UI.
Пример:
И
false * true = false
ИЛИ
false + true = true
Штрих Шеффера
false | true = true
Исключающее ИЛИ
false xor true = true
